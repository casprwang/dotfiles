# Dotfiles

![image](https://cloud.githubusercontent.com/assets/19645990/22623539/80e25f6a-eb2c-11e6-800c-6abc7bcdbb6c.png)
## Configurations for:
* Editors
  * Vim (light weighted, simple, fast)
  * Neovim (asynchronous, powerful)
  * Atom (powerful, compatible with vim)
* Terminal tools
  * iTerm
  * Tmux
  * Zsh
* Macros
  * Hammerspoon 
  * Alred 3
* Linting 
  * Eslint
* New tools
